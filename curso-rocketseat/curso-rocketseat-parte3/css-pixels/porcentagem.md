# Porcentagem (%)

* Relativo a algum valor

```css
html {
    font-size: 50%
}
/* nesse caso, o 50% vai ser relativo ao que está de padrão no navegador (16 px) */
```

```html
<div></div>
```

```css
div {
    background-color: blue;
    width: 100%;
    height: 100px;
} 
/* 100% do body, que é o elemento pai */
```
